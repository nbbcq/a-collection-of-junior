相关函数位于 src 目录下，测试代码位于 tests 文件夹下.

* 3-test.c -- 在屏幕上输出一个整数的二进制表示
* 4-test.c -- 将一个 16 bits 整数转化为 BCD 码表示(十六进制输出)
* 5-test.c -- 检测当前计算机是大端机还是小端机
